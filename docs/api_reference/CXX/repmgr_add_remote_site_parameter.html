repmgr_add_remote_site

Adds a new replication site to the Replication Manager's list of known sites.

The syntax of this parameter in the DB_CONFIG file is a single line with the string repmgr_add_remote_site, one or more whitespace characters, and the host, port and optional peer parameters specified. Where host is the site's host identification string, generally a TCP/IP host name; port is the port number on which the remote site is receiving. For example: 

repmgr_add_remote_site exmample.com 49201 peer

For more information, see DbEnv::repmgr_add_remote_site().
